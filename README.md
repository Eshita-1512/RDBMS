# SkillBridge

A comprehensive learning management system built with FastAPI and React.

## Project Overview

SkillBridge is a modern platform for managing courses, assignments, and student enrollments with a robust backend API and an intuitive frontend interface.

## Project Structure

- **backend/** - FastAPI application with REST APIs
- **frontend/** - React application for the user interface
- **docs/** - Project documentation

## Getting Started

### Prerequisites
- Docker and Docker Compose
- Python 3.9+ (for backend development)
- Node.js 16+ (for frontend development)

### Running with Docker Compose

```bash
docker-compose up
```

This will start both the backend and frontend services.
